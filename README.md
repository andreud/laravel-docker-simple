# Run containers

	$ docker compose up

# Install with new Laravel instance: 

	cd application

	laravel new laravel

	(if "laravel" command not in path)
	/root/.composer/vendor/laravel/installer/bin/laravel new laravel

	cd laravel
	nvm i 22
	npm install && npm run build
	composer run dev


	The [pcntl] extension is required to run Pail.
